# Тарифные планы Eldoleado — Концепция

> **Цель:** Разные модели использования платформы для разных типов бизнеса

---

## План 1: Индивидуальный (Solo)

### Для кого
- Одиночка, самозанятый, ИП
- Малый бизнес без сотрудников
- **Примеры:** мастер маникюра, психолог, сантехник-фрилансер, репетитор

### Архитектура
- **Пользователи:** 1 человек
- **Тенанты:** нет
- **База данных:** облачная (shared)
- **Инфраструктура:** Eldoleado

### Особенности
- Простая настройка через AI-конструктор
- Подключение каналов **через маркетплейс**
  - Арендует WhatsApp API, Telegram и другие каналы за фикс
  - Не нужно разбираться с API-ключами
- Полный доступ к блокам конструктора
- Быстрый старт — за один вечер

### Оплата
- Владелец → Eldoleado (подписка)
- Дополнительно: каналы через маркетплейс (фикс за канал)

### Ограничения
- Один пользователь
- Нет подразделений, групп, ролей
- Shared инфраструктура

## План 2: Агентский (Agency)

### Для кого
- Фрилансеры, ведущие клиентов
- Небольшие агентства (маркетинг, SMM, SEO)
- **Примеры:** авитолог с 100 клиентами, SMM-агентство, SEO-менеджер

### Архитектура
- **Пользователи:** 1 агент + 50-200 тенантов (клиентов)
- **Тенанты:** есть (клиенты агента)
- **База данных:** облачная (shared с изоляцией)
- **Инфраструктура:** Eldoleado

### Особенности
- **Мультитенантность:** каждый клиент — отдельный тенант
- **Центральный дашборд:** сводная аналитика по всем клиентам
- **Шаблоны вертикалей:** настроил один раз → развернул на 50 клиентов
- **Гибкий доступ клиентов (настраивается агентом):**
  - Вариант А: клиенты работают через агента (он единственный админ)
  - Вариант Б: клиенты имеют свой вход и управляют своей частью
- Переключение между тенантами одним кликом

### Сценарии использования

**Авитолог:**
- 100 клиентов, у каждого свой аккаунт Авито
- Отслеживает лиды с объявлений всех клиентов
- Видит конверсию по каждому
- Настраивает автоответы, возврат холодных лидов
- Запускает/останавливает объявления
- Сводная аналитика эффективности

**SEO-менеджер:**
- 50 сайтов в управлении
- Мониторинг позиций, трафика
- Лиды с форм обратной связи
- Воронки для каждого клиента
- Отчёты по конверсии

**SMM-агентство:**
- 80 бизнесов в соцсетях
- Лиды из VK, Instagram, Telegram
- Автогруппы по активности
- Запуск/остановка рекламы
- Аналитика охватов

### Оплата
- Агент → Eldoleado (фикс за платформу)
- Агент сам решает как монетизировать своих клиентов

### Ограничения
- Лимит тенантов: 50-200 (зависит от тарифа)
- Shared инфраструктура

## План 3: Энтерпрайз (Enterprise)

### Для кого
- Крупные компании, корпорации
- Сети с филиалами
- **Примеры:** сеть сервисных центров (50 филиалов), банк, ритейл-сеть

### Архитектура
- **Пользователи:** множество сотрудников
- **Тенанты:** нет (вместо этого **подразделения, филиалы, группы**)
- **База данных:** self-hosted (своя инфраструктура)
- **Инфраструктура:** клиента

### Особенности
- **Корпоративная структура:**
  - Отделы, филиалы, подразделения
  - Иерархия ролей и прав доступа
  - Группы сотрудников
- **Полный контроль над данными**
- Свои настройки безопасности, бэкапы
- Интеграция с корпоративными системами:
  - Active Directory
  - 1С, SAP
  - Внутренние системы
- Кастомизация под процессы компании
- SLA техподдержки

### Развёртывание

**Базовый вариант:**
- Клиент сам разворачивает на своей инфраструктуре
- Документация + техподдержка

**Премиум вариант:**
- Eldoleado настраивает за доп. плату
- Внедрение, обучение персонала, интеграции

### Обновления
*В процессе обдумывания (Docker образы, Git, автообновления)*

### Оплата
- Фиксированная лицензия (за платформу)
- Опционально: внедрение, обучение, поддержка

### Ограничения
- Требует собственную инфраструктуру
- Самостоятельная поддержка или за доп. плату

## План 4: Коммерческая вертикаль (Commercial Vertical)

### Для кого
- Предприниматели, создающие свою CRM-услугу на базе Eldoleado
- Франшизы, отраслевые решения
- **Примеры:** "CRM для стоматологий", "CRM для автосервисов", "CRM для фитнес-клубов"

### Архитектура
- **Пользователи:** владелец вертикали + их клиенты
- **Тенанты:** есть (клиенты владельца вертикали)
- **База данных:** облако или self-hosted (на выбор)
- **Инфраструктура:** Eldoleado или клиента

### Особенности
- **White label:** полный ребрендинг под свой бренд
- **Собственный биллинг:** владелец сам устанавливает цены
- **Тарифные планы внутри вертикали:**
  - Владелец создаёт свои тарифы (лайт, стандарт, премиум)
  - Управляет доступом к блокам на каждом тарифе
- **Гибкая настройка прав доступа**
- **API для интеграций**

### Сценарии

**"CRM для стоматологий":**
- Создатель продаёт 500 клиникам
- Каждая клиника — тенант
- Свои тарифы: "Базовый", "Расширенный", "Премиум"
- Брендинг: логотип клиники, не Eldoleado

**"CRM для автосервисов":**
- Франшиза на 200 СТО
- Единая система для всей сети
- Центральная аналитика по франшизе

### Монетизация
- Владелец вертикали платит Eldoleado (% или фикс)
- Он сам устанавливает цены для своих клиентов
- *Детали модели оплаты — в проработке*

### Возможности
- Полная кастомизация под нишу
- Создание своей экосистемы
- Собственный маркетплейс расширений (опционально)

### Ограничения
- *В процессе проработки*

## План 5: Маркетплейс (интеграционный слой)

### Суть
Централизованный маркетплейс каналов, интеграций и MCP-расширений

### Владелец
**Eldoleado** (модель как Apple App Store)

### Что в маркетплейсе

**1. Каналы связи:**
- WhatsApp API
- Telegram Bot API
- VK API
- Avito API
- Голосовые операторы
- Email-интеграции

**2. Интеграции:**
- Службы доставки (СДЭК, Boxberry)
- Эквайринг (ЮKassa, Tinkoff)
- 1С, SAP
- Операторы фискальных данных (ОФД)
- Карты (Яндекс, Google, 2ГИС)

**3. MCP-расширения:**
- Кастомная бизнес-логика
- Специфичный функционал для ниш

**4. Готовые вертикали (шаблоны):**
- "CRM для салона красоты"
- "CRM для сервисного центра"
- И другие

### Типы MCP

**Неверифицированные:**
- Только для личного использования
- Не проходят проверку
- Риск на совести владельца
- Нельзя выложить на маркетплейс

**Верифицированные:**
- Подаются в виде исходного кода
- Eldoleado проверяет на:
  - Бэкдоры
  - Вредоносный код
  - Соответствие стандартам безопасности
- После проверки попадают в маркетплейс
- Доступны всем пользователям

### Как работает для разных планов

**План 1 (Индивидуальный):**
- Арендует канал (например, WhatsApp API) за фикс
- Маркетплейс → API поставщика → вертикаль клиента
- Не нужно разбираться с техническими деталями
- Оплата: Eldoleado (подписка) + маркетплейс (каналы)

**План 2-4 (Агентский, Энтерпрайз, Коммерческая вертикаль):**
- Могут арендовать через маркетплейс
- Или подключить свои API-ключи напрямую
- Гибкий выбор

### Монетизация маркетплейса
- Eldoleado берёт % с продаж MCP (например, 30/70 как Apple)
- Для каналов — фикс или % от использования
- Разработчики MCP получают свою долю

### Безопасность
- Все верифицированные MCP проходят проверку
- Регулярный аудит безопасности
- Быстрая реакция на уязвимости
- Рейтинговая система для разработчиков

## Ключевые отличия планов

### Индивидуальный
**Суть:** "Я сам себе CRM настраиваю"
- Максимально простой вход
- Не нужно разбираться в технических деталях
- AI-конструктор всё соберёт за 10-15 минут

### Агентский
**Суть:** "Я управляю CRM для моих клиентов"
- Единый вход для агента
- Клиенты могут даже не знать про Eldoleado
- Шаблоны для быстрого масштабирования

### Энтерпрайз
**Суть:** "Наша CRM на нашей инфраструктуре"
- Полный контроль над данными
- Корпоративная безопасность
- Интеграция с существующими системами

### Коммерческая вертикаль
**Суть:** "Создаю свой CRM-бизнес на базе Eldoleado"
- Свой бренд, свои правила
- Монетизация клиентов
- Eldoleado как платформа, не конкурент
