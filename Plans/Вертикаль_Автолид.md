# Вертикаль "Автоклиент" — Универсальная система продаж

> **Концепция:** Универсальный инструмент для привлечения и конверсии клиентов, работающий для ЛЮБОГО бизнеса с продажами.

---

## Суть вертикали

**Не CRM для конкретной ниши. Система продаж для всех.**

### Что делает:
1. **Собирает лиды** — Telegram-бот, лендинг, формы
2. **Квалифицирует** — AI задаёт вопросы, определяет горячесть
3. **Прогревает** — Email-серии, напоминания, follow-up
4. **Конвертирует** — Воронка продаж, звонки, сделки
5. **Анализирует** — CAC, LTV, конверсия, ROI

---

## Для кого работает

### Примеры ниш:
- Сервисные центры (продажа ремонта)
- Скупки/Комиссионки (привлечение продавцов)
- Салоны красоты (запись на услуги)
- Репетиторы (поиск учеников)
- Агентства (продажа услуг)
- Стоматологии (запись на приём)
- Фитнес-клубы (продажа абонементов)
- **И даже для продажи самих вертикалей!**

## Блоки вертикали "Автоклиент"

### 1. Блок Коммуникаций
**Сбор лидов из всех каналов:**
- Telegram-бот (квалификация через вопросы)
- Лендинг с формами
- WhatsApp (входящие запросы)
- Email (рассылки)

### 2. Блок AI Контекста
**Квалификация лидов:**
- AI задаёт уточняющие вопросы
- Извлекает: потребность, бюджет, срочность
- Скоринг: hot/warm/cold
- Автораспределение по менеджерам

### 3. Блок Канбан
**Воронка продаж:**
- Статусы: Лид → Квалификация → Звонок → Предложение → Сделка
- Drag-and-drop карточек
- Быстрые действия
- Интеграция с AmoCRM/Bitrix24 (на старте)

### 4. Блок Триггеров
**Автоматизация:**
- Email-серии (день 0, 2, 4, 7, 14)
- Напоминания менеджеру (позвонить через 2 дня)
- Реактивация холодных лидов
- Follow-up после звонка

### 5. Блок Уведомления
**Мультиканальные рассылки:**
- Email (SendPulse, UniSender)
- SMS (Twilio, SMS.ru)
- Push (Firebase)
- WhatsApp/Telegram

### 6. Блок Календарь
**Запись на звонки/встречи:**
- Интеграция с Calendly
- Автоматические напоминания
- Синхронизация с Google Calendar

### 7. Блок Аналитика
**Метрики продаж:**
- Воронка конверсий (сколько на каждом этапе)
- CAC (стоимость привлечения клиента)
- LTV (пожизненная ценность)
- ROI по каналам (РСЯ, контент, сарафан)

## Тарифные планы

### Тариф "Старт"
**Для кого:** Малый бизнес, начинающие предприниматели

**Что включено:**
- Ограниченное количество лидов
- Минимальное количество пользователей
- Telegram-бот + Email-рассылки
- Базовая аналитика
- Готовые шаблоны (email, боты)

---

### Тариф "Рост"
**Для кого:** Растущий бизнес, агентства

**Что включено:**
- Увеличенный лимит лидов
- Несколько пользователей
- WhatsApp + SMS интеграции
- Продвинутая аналитика (CAC, LTV, воронки)
- A/B тесты
- Кастомные шаблоны

---

### Тариф "Премиум"
**Для кого:** Крупный бизнес, франшизы, владельцы коммерческих вертикалей

**Что включено:**
- Безлимит лидов
- Безлимит пользователей
- White label (свой бренд)
- API доступ
- Приоритетная поддержка
- Персональный онбординг

Включается в нишевые вертикали (Ремонт, Скупка, Салоны).

**Пример:**
- Вертикаль "Ремонт": базовая цена
- + "Автоклиент": доплата
- **Bundle-скидка:** при покупке вместе

## Стратегия продаж

### Cross-sell (допродажа)
**Сценарий:**
1. СЦ покупает вертикаль "Ремонт"
2. Через месяц видит результаты
3. Предлагаешь добавить "Автоклиент"
4. Теперь СЦ активно ищет клиентов, а не только обрабатывает входящие

**Результат:** Увеличение среднего чека через допродажу

---

### Upsell (повышение тарифа)
**Сценарий:**
1. СЦ покупает "Автоклиент"
2. Через месяц видит, что лиды идут
3. Предлагаешь добавить вертикаль "Ремонт"
4. Теперь СЦ не только привлекает, но и обрабатывает клиентов

**Результат:** Увеличение среднего чека через допродажу

---

### Bundle (пакет)
**Сценарий:**
1. СЦ покупает сразу "Ремонт" + "Автоклиент"
2. Скидка за пакет
3. Выше вероятность покупки (всё сразу решено)

**Результат:** Более высокий средний чек с первого дня

## Преимущества "Автоклиента"

### 1. Универсальность
Работает для ЛЮБОГО бизнеса с продажами (не только СЦ, салоны, скупки).

### 2. Моментальная ценность
В первую неделю клиент видит результаты:
- Telegram-бот собирает лиды
- AI квалифицирует их
- Email-серия прогревает
- Напоминания на звонки

### 3. Viral loop
Один довольный клиент → 3-5 новых (из разных ниш)

### 4. Доказательство платформенности
Для инвесторов: не нишевый продукт, а универсальная платформа.

### Позиционирование:
**"Автоматизируйте продажи с AI. Для любого бизнеса."**
