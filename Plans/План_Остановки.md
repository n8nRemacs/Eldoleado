## Ключевые инсайты

### Vertical SaaS Factory — что это:
```
Shopify = e-commerce для всех
Canva = дизайн для всех
Eldoleado = CRM/автоматизация для всех (через AI-диалог)
```

### Почему это работает:
- Демократизация enterprise-технологий
- AI убирает барьер технической сложности
- Пользователь описывает бизнес → получает готовую систему

### Блоки собранные в демо-диалоге:
1. Коммуникаций (WhatsApp, VK, Instagram)
2. AI Контекста (извлечение даты, адреса, кол-ва)
3. Триггеров (напоминания, автодействия)
4. Уведомления
5. Лояльности (каждый 5-й заказ -10%)
6. Платежей (Сбербанк, наличные)
7. Канбан (статусы заказов)
8. Аналитика (еженедельный отчёт)
