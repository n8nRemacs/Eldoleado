# Техническое задание: MVP Eldoleado (2 недели)

> **Цель:** Рабочий прототип для демонстрации потенциальным клиентам
> **Срок:** 2 недели (14 дней)
> **Дата начала:** 16 декабря 2025
> **Дата сдачи:** 30 декабря 2025

---

## 1. Общее описание

### 1.1 Что делаем

Омниканальный мессенджер для сервисных центров, объединяющий все каналы коммуникации в одном интерфейсе.

### 1.2 Что НЕ делаем в MVP

- AI-подсказки и автоответы
- Контекст-машина
- Аналитика и отчёты
- Интеграции с CRM (HelloClient, LiveSklad)
- Поиск запчастей
- Транскрибация звонков

### 1.3 Результат MVP

Пользователь может:
1. Получать сообщения из Telegram, WhatsApp, Авито, Max в одном окне
2. Отвечать на сообщения из любого канала
3. Видеть историю переписки с клиентом
4. Экспортировать сообщения в JSON
5. Работать через веб-интерфейс и Android-приложение

### 3.4 Android-приложение

#### 3.4.1 Два режима работы

| Режим | Вход | Описание |
|-------|------|----------|
| **Чат** | Обычный пароль | Клиентский интерфейс (WebView на веб) |
| **Сервер** | Специальный пароль | Фоновый туннель-сервис |

#### 3.4.2 Режим "Чат"

- WebView загружает веб-интерфейс
- Push-уведомления о новых сообщениях
- Стандартное приложение для работы

#### 3.4.3 Режим "Сервер"

- Минималистичный интерфейс
- Статус подключения к серверу
- Статус каналов (WhatsApp, Авито, Max)
- Кнопка "Свернуть в фон"
- Работает как Android Service
- Автозапуск при загрузке устройства

## 5. База данных

### 5.1 Основные таблицы

```sql
-- Тенанты (клиенты системы — СЦ)
CREATE TABLE tenants (
    id UUID PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT NOW()
);

-- Пользователи (сотрудники СЦ)
CREATE TABLE users (
    id UUID PRIMARY KEY,
    tenant_id UUID REFERENCES tenants(id),
    email VARCHAR(255) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    name VARCHAR(255),
    role VARCHAR(50) DEFAULT 'operator',
    created_at TIMESTAMP DEFAULT NOW()
);

-- Каналы (подключения к мессенджерам)
CREATE TABLE channels (
    id UUID PRIMARY KEY,
    tenant_id UUID REFERENCES tenants(id),
    type VARCHAR(50) NOT NULL, -- telegram, whatsapp, avito, max
    status VARCHAR(50) DEFAULT 'disconnected',
    credentials JSONB, -- токены, сессии (зашифровано)
    settings JSONB,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- Клиенты (покупатели СЦ)
CREATE TABLE clients (
    id UUID PRIMARY KEY,
    tenant_id UUID REFERENCES tenants(id),
    name VARCHAR(255),
    phone VARCHAR(50),
    email VARCHAR(255),
    metadata JSONB,
    created_at TIMESTAMP DEFAULT NOW()
);

-- Привязки клиентов к каналам
CREATE TABLE client_channels (
    id UUID PRIMARY KEY,
    client_id UUID REFERENCES clients(id),
    channel_type VARCHAR(50) NOT NULL,
    channel_identifier VARCHAR(255) NOT NULL, -- telegram_id, phone, avito_user_id
    created_at TIMESTAMP DEFAULT NOW(),
    UNIQUE(client_id, channel_type, channel_identifier)
);

-- Диалоги
CREATE TABLE dialogs (
    id UUID PRIMARY KEY,
    tenant_id UUID REFERENCES tenants(id),
    client_id UUID REFERENCES clients(id),
    channel_id UUID REFERENCES channels(id),
    channel_chat_id VARCHAR(255), -- внешний ID чата
    status VARCHAR(50) DEFAULT 'active',
    last_message_at TIMESTAMP,
    unread_count INT DEFAULT 0,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- Сообщения
CREATE TABLE messages (
    id UUID PRIMARY KEY,
    dialog_id UUID REFERENCES dialogs(id),
    direction VARCHAR(20) NOT NULL, -- incoming, outgoing
    content_type VARCHAR(50) DEFAULT 'text', -- text, image, file
    text TEXT,
    media_url VARCHAR(500),
    external_id VARCHAR(255), -- ID в мессенджере
    status VARCHAR(50) DEFAULT 'sent', -- sent, delivered, read, failed
    metadata JSONB,
    created_at TIMESTAMP DEFAULT NOW()
);

-- Туннели (подключённые телефоны)
CREATE TABLE tunnels (
    id UUID PRIMARY KEY,
    tenant_id UUID REFERENCES tenants(id),
    device_id VARCHAR(255) UNIQUE NOT NULL,
    device_name VARCHAR(255),
    status VARCHAR(50) DEFAULT 'offline',
    last_seen_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW()
);

-- Индексы
CREATE INDEX idx_dialogs_tenant ON dialogs(tenant_id);
CREATE INDEX idx_dialogs_updated ON dialogs(updated_at DESC);
CREATE INDEX idx_messages_dialog ON messages(dialog_id);
CREATE INDEX idx_messages_created ON messages(created_at DESC);
CREATE INDEX idx_client_channels_identifier ON client_channels(channel_identifier);
```

## 6. API

### 6.1 REST API endpoints

#### Авторизация

```
POST /api/auth/login
POST /api/auth/logout
POST /api/auth/refresh
GET  /api/auth/me
```

#### Диалоги

```
GET    /api/dialogs                 # Список диалогов

GET    /api/dialogs/:id/messages    # Сообщения диалога
POST   /api/dialogs/:id/messages    # Отправить сообщение
PUT    /api/dialogs/:id/read        # Пометить прочитанным
```

#### Каналы

```
GET    /api/channels                # Список каналов
POST   /api/channels                # Добавить канал
PUT    /api/channels/:id            # Обновить канал
DELETE /api/channels/:id            # Удалить канал
POST   /api/channels/:id/connect    # Подключить
POST   /api/channels/:id/disconnect # Отключить
```

#### Экспорт

```
GET    /api/export/dialogs          # Экспорт всех диалогов
GET    /api/export/dialogs/:id      # Экспорт одного диалога
```

## 9. Риски и митигация

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Блокировка WhatsApp | Средняя | Высокое | Туннель через IP клиента |
